# How to Install Sambahvl Plugin

At first, download sambahvl zip file: https://github.com/zekiahmetbayar/liman-sambahvl/archive/refs/heads/main.zip

Go to your liman web interface and go to "Sistem Ayarları" section from left menu.

<img src="./Assets/sistemayarları.png">

Go to 'Eklentiler' tab.

Click "Yükle" button and choose the zip file that you have downloaded.

You have loaded sambahvl plugin to liman.
